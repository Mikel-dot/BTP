# ⚡ BTP - Best Trick Pony 
> **The Math-Based Draft Assistant & Self-Coaching Tool for League of Legends.**

![Status: Beta](https://img.shields.io/badge/Status-Beta_v0.8-00f2ff?style=for-the-badge)
![Built with: HTML/CSS/JS](https://img.shields.io/badge/Built_with-HTML%20%7C%20CSS%20%7C%20JS-00f2ff?style=for-the-badge)
![Privacy: 100% Local](https://img.shields.io/badge/Privacy-100%25_Local-0ac8b9?style=for-the-badge)

BTP is a lightweight, non-intrusive desktop assistant designed for players who want to master their champion pool through empirical data and self-awareness. Unlike AI-driven tools, BTP focuses on **pure mathematics** and **mindfulness**, helping you make better decisions without playing the game for you.

---

## 🚀 Key Features

* 📊 **Mathematical Draft Assistant:** Real-time pick recommendations based on your personal champion pool and high-ELO statistical deltas.
* 🧠 **Self-Coaching Tracker:** Automatic post-game popups to grade your performance (Vision, CS, Map Awareness) while the game is still fresh.
* 🛡️ **Privacy First:** No cloud databases. All your notes, settings, and API keys are encrypted and stored locally on your machine.
* 🔌 **LCU Integration:** Seamlessly detects champion select and game end using the official League Client Update protocol.
